# Simple-Calculator
A simple calculator app using TextViews and Buttons which can perform five basic Arithmetic Operations.

## Available Arithmetic Operations 
<table>
  <tr>
    <td> Addition </td> 
    <td align="center"> + </td>
  </tr>
  <tr>  
    <td> Subtraction </td> 
    <td align="center"> - </td>
  </tr>
  <tr>
    <td> Multiplication </td> 
    <td align="center"> * </td>
  </tr>
  <tr>
    <td> Division </td> 
    <td align="center"> / </td>
  </tr>
  <tr>
    <td> Modulus </td> 
    <td align="center"> % </td>
  </tr>
</table>
